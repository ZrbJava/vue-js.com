# vue-js.com

> 初学vuejs，尝试重构vue社区，此项目参照了 bailicangdu 的 vue2-elem（https://github.com/bailicangdu/vue2-elm） 项目，谢谢 @bailicangdu

## 技术栈
vue2 + vuex + vue-router + webpack + ES6/7 + fetch 

## 运行项目
``` bash

npm install

npm run dev

npm run build

npm run build --report
```
